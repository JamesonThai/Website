Chronic Stroke Lesion Detection

September 2018 - Present

This project aims to automate lesion detecthion and analysis in a chronic stroke patient's MRI scans. Using Random Forests and neighborhood analysis of regions near the cortex, we can reduce our search area and increase the probability of detection lesions. Our dataset of 300 3D MRI scans have been verified and labeled image by image by a certified neurologists. Through this application, we aim to give neurologists a more informed decision that not only improves accuracy and treatment, but also helps patients recieve treatment sooner.

Quick summary

• Formulate automatic segmentation model based on Random Forests and Neighborhood algorithms for testing nearly 300 3D MRI images.

• Image segmentation and splicing of 3D images into 2D.

More information about this project will be posted as the website is developed.

Amazon Review Ratings Recommender

April 2018 - Present

Using an Amazon data set of over 12 million reviews, my team and I parsed, filtered, and trained Python models to recommend ratings for a user's review. We have currently achieved an accuracy of 63% however we are improving this by reducing filler words and trying faster and more efficient models. At the same time we are visualizing this data to understand what goes into these reviews in the context of word usuage.

Quick summary

• Models trained: KNN, Naïve Bayes, Logistic Regression, Doc2Vec, Decision Trees, and Linear SVM

• Technologies Used: Jupyter Notebook, Keras, Pandas, matplotlib, plotly and ggplot.

More information about this project will be posted as the website is developed.
